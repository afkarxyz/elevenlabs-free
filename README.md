## üñ•Ô∏è Installation

1. 
   ```bash
   pip install flask elevenlabs
   ```
   
2. 
   ```bash
   git clone https://github.com/afkarxyz/elevenlabs-free.git
   cd elevenlabs-free
   ```
   
3. 
   ```bash
   python app.py
   ```

4. Access the web interface at `http://127.0.0.1:5000`
5. Enter your `API key` in the API Key tab
